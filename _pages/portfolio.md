## UC Davis AI Academic Advisor

[pathfinder.demo.richis.top](https://pathfinder.demo.richis.top)

An agentic RAG advising system for UC Davis course planning. The project combines function calling, structured academic data, schedule-building workflows, and conversation summarization so students can explore degree paths and course plans in a more personalized way.

**Core work:** knowledge-base construction from 9,000+ course records and 10,000+ student reviews, data normalization, multi-turn context compression, API integration, and export/synchronization workflows.

## Multi-Modal Route Planning System

A C++ route planning system built on OpenStreetMap and transit network data. It computes practical routes across walking, biking, and bus options while accounting for one-way roads, speed limits, and large graph performance.

**Core work:** Dijkstra's algorithm, Expat XML parsing, graph storage optimization, PImpl-based architecture, navigation instruction generation, Makefiles, GoogleTest, and Google Mock.
