# Ruizhi Jin Academic Homepage

Personal academic homepage based on the Academic Pages Jekyll template.

## Local preview

Install Ruby and Bundler, then run:

```bash
bundle install
bundle exec jekyll serve
```

The site includes:

- About page
- Projects page
- CV page with downloadable CV
- Photography wall

## Public deployment

Recommended GitHub Pages setup:

1. Rename this repository on GitHub to `ruizhijin-coder.github.io`.
2. In repository settings, go to `Pages`.
3. Set `Build and deployment` to `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save. The public site should become available at:

```text
https://ruizhijin-coder.github.io/
```

For a custom domain later, configure it in GitHub Pages settings first, then
add the DNS records through the domain provider.
